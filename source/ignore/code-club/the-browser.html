HTML ‚Äì Content Design ‚Äì What it says üí¨

HTML describes the type of information being displayed within a document. If we're starting a new paragraph then we should use a <p> element. If we're starting a new section we can create a header with an <h1> element. Each element must also be closed so you'll end up with something like this ‚Äì 

<h1>Fresh Prince</h1> 
 <p>Now this is a story all about how my life got flipped turned upside down</p>

This is great because 1. It tells the browser how to decorate the document and 2. It allows other devices to access this material more easily. We have to take into account Screen Readers, Accessibility Shortcuts and other assistive devices like voice assistants.

Caption competition

Here's the first task:

  • Go to a website that you use often
  • Find some text on the page
  • Change the text to make it silly and/or funny

When you're browsing the web this week, look at websites and have a think about some simple things you might like to change: try changing the colours of some elements; try deleting GDPR notices; try removing ads; try changing the typography so that it reads better; try making a silly headline.

When you're done, post your creations in the #code_club Slack group!
